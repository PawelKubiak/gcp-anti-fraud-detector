# gcp-anti-fraud-detector

## Prerequisites

* Active Google Cloud Platform account: https://cloud.google.com/free/ (you need to attach Credit Card in order to use trial period)
* Some Java and Maven knowledge
* Own laptop with:
  * wi-fi
  * newest Chrome browser
